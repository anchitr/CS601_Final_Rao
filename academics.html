Boston University

I've taken six courses so far at Boston University. Here are some of the courses I've taken and my thoughts on them.

Class Code Class Name Professor Difficulty Rating Comments
MET-CS 669 Database Design and Implementation for Business Warren Mansur 4.1/5.0 Challenging course but highly informative and useful in industry.
MET-CS 521 Information Structures with Python Jos Purvis 3.1/5.0 I already came into the class knowing Python due to my professional background. It was very easy for me.
MET-CS 625 Business Data Communication and Networks Maryan Rizinski 4.6/5.0 By far my most difficult classes given how little I knew about networking. Assignments took a long time.
MET-CS 546 Introduction to Probability and Statistics Anatoly Temkin 3.6/5.0 Mathematically challenging, but essential concepts to understand for the field of DS/ML.
MET-CS 544 Foundations of Analytics and Data Visualization Suresh Kalathur 4.3/5.0 R is a difficult language to learn, but the professor was very helpful and the assignments were very useful.
MET-CS 601 Web Application Development Christian Hur 3.2/5.0 Very quickly covers the basics, but the assignments allow for a lot of creativity and learning.
